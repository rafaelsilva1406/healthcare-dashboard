# Rsbuild/React/Bootstrap UI project
- Bootstrap for UI
- React for application
- Actual bootstrap loader rsbuild
    
## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
